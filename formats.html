Формати

УниРанг може да се използва с 3 формата (.json, .xls, .xlsx)

Какво е JSON и как да го използвам в УниРанг?

JSON е независим формат на данни, който не е обвързан с езика JavaScript. В същото време JSON е идеалният формат за обмен на данни между системи със своята проста структура и кодиране в набора от знаци Unicode. Винаги се обменя между приложенията като цяло. Това е най-лесният за човешка обработка на JSON документ, ако документът е наличен като файл във файловата система. Тук обикновено се използва файловото разширение „.json“. Съдържанието на JSON документ е основно обектно-ориентирано по структура.

За JSON файла на кандидатите структурата за всеки студент трябва да бъде:
{
"UIN": ...,
"firstScore": ...,
"course_id1": ...,
"secondScore": ...,
"course_id2": ...,
"thirdScore": ...,
"course_id3": ...,
"forthScore": ...,
"course_id4": ...,
"fifthScore": ...,
"course_id5": ...
}
За JSON файла на университетите структурата за всеки университет трябва да бъде:
{
"speciality": ...,
"free_places": ...,
"university": ...,
"course_id": ...,
"university_id": ...,
}

Какво е XLS и XLSX и как да го използвате в УниРанг?

XLSX файловете са файлове, използвани в Microsoft Excel, приложение за електронни таблици, което използва таблици за организиране, анализиране и съхраняване на данни. Всяка клетка може да съдържа текстови или числови данни, включително включване на математически формули. Въведен за първи път с Microsoft Office 2007 в стандарта Office Open XML, XLSX е форматът по подразбиране при създаване на електронна таблица с помощта на модерни версии на Excel. Форматът XLXS замени формат XLS, който Excel използваше преди.

За XLS или XLSX файла с кандидати таблицата трябва да има следните колони:
UIN
firstScore
course_id1
secondScore
course_id2
thirdScore,
course_id3
forthScore
course_id4
fifthScore
course_id5
За университетския XLS или XLSX файл таблицата трябва да има такива колони:
speciality
free_places
university
course_id
university_id
